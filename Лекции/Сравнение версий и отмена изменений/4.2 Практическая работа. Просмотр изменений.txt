пше вшааgit log - просмотреть историю коммитов и просмотр хэшей
git ststus - просмотреть все незакомиченные изменения
git diff - просмотреть изменения всех файлов
git diff index.html - посмотреть какие изменения были у файла index.html
git diff (хэш коммита более раннего(или первые 4 символа)) (хэш коммита более позднего (или первые 4 символа)) index.html - просмотр разницы между коммитами конкретного файла.
если мы поменяем коммиты местами то изменения будут отображаться в обратном порядке.
git diff index.html index2.html - так можно сравнивать два файла которые лежат даже вне репозитория.
git blane index.html - посмотреть в рамках каких коммитов, каким пользователем, когда, и какие конкретно изменения вносились
